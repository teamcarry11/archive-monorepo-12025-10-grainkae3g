## Key Features of Unstoppable Web3 Domains

According to [Unstoppable Domains support](https://support.unstoppabledomains.com/support/solutions/articles/48001181690-what-are-web3-domains-):

**True Ownership**:
- Stored as NFTs in your crypto wallet
- No renewal fees (except .eth)
- Can't be taken away by centralized authority
- Yours for life, no middlemen

**Crypto Transactions**:
- Replace long wallet addresses with human-readable names
- Support 300+ cryptocurrencies
- Example: `matt.crypto` instead of `0x742d35Cc6634C0532925a3b844Bc454e4438f44e`

**Web3 Login (SSO)**:
- Single sign-on for dApps and Web3 platforms
- Portable online identity you control
- Similar to "Login with Google" but decentralized
- [Login with Unstoppable](https://support.unstoppabledomains.com/support/solutions/articles/48001215516-what-is-login-with-unstoppable-)

**Web3 Websites**:
- Host content on IPFS (decentralized storage)
- Resolve natively in Brave and Opera browsers
- No centralized hosting required

**Supported Blockchains**:
- Polygon
- Ethereum (ERC-721)
- Solana ([currently custodial, self-custody coming August](https://support.unstoppabledomains.com/support/solutions/articles/48001272836-accessing-your-solana-domains))
- Base
- Sonic

**Web3 Profiles & Identity**:
- UD.me profiles showcasing digital assets
- Social links, pfp, onchain achievements
- Secure end-to-end encrypted messaging

## UP.io - Self-Custody Wallet with MPC Security

According to [Unstoppable Domains UP.io documentation](https://support.unstoppabledomains.com/support/solutions/articles/48001269485-what-is-up-io-):

**What is UP.io?**  
UP.io is a next-generation self-custody wallet providing full control over digital assets on any device. Built with Multi-Party Computation (MPC) technology for top-tier security and ease of use, eliminating traditional private keys or seed phrases.

**Authentication**:
- Simple email and password login (no seed phrases!)
- Password-based authentication system
- Emergency backup recovery (set new password if forgotten)
- Each backup invalidates previous ones for security

**How MPC Works**:
1. **Password-Based Authentication** - Login with email/password
2. **Key Shares Computed Separately** - Multiple cryptographic shares generated
3. **MPC Generates Signature** - All shares participate without combining
4. **Secure Execution** - No single entity (including UP.io) has full key access

**Security Benefits**:
- ✅ **No Single Point of Failure** - Keys never exist in full form
- ✅ **Distributed Signing** - Multiple key shares required (hacker needs all)
- ✅ **Tamper-Resistant** - One compromised share cannot sign alone
- ✅ **100% User Control** - Only you approve transactions
- ✅ **Fireblocks MPC** - One share on device, one with Fireblocks (neither alone suffices)

**Supported Blockchains**:
- Ethereum
- Polygon
- Base
- Solana
- Bitcoin

**Platform Availability**:
- Web application
- Browser extension
- iOS mobile app
- Android mobile app

**Key Differences vs Custodial**:
- UP.io = Self-custody (you control keys via MPC)
- Unstoppable Lite = Custodial (UD manages keys)
- Both use Fireblocks MPC technology

**Critical Warning**:
- Your password IS your key
- Losing password = irreversible asset loss
- MUST create emergency backup
- Unstoppable Domains cannot recover if no backup
