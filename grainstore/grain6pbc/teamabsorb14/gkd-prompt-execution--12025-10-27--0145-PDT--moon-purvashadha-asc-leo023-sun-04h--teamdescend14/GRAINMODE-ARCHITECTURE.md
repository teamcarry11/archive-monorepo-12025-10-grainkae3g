## What is a Grainmode?

A grainmode is a lens through which the same grain can be experienced. Like looking at a diamond from different angles, each mode reveals different facets of the same essential truth. The content remains identical in meaning; the voice, metaphors, and presentation shift to match different learning styles and contexts.

**Core Principle**: One grain, multiple modes. Same knowledge, different doors.

## Technical Implementation

### Grainmode Metadata

Each grain can have multiple mode files:
```
xbdghj-grainbranch-readme-sync-ketos.md          (default: Glow G2)
xbdghj-grainbranch-readme-sync-ketos-trish.md    (Trish B1 mode)
xbdghj-grainbranch-readme-sync-ketos-helen.md    (Helen Atthowe mode)
xbdghj-grainbranch-readme-sync-ketos-davinci.md  (Da Vinci cipher)
xbdghj-grainbranch-readme-sync-ketos-ariana.md   (Ariana Grande mode)
```

### Grainmode Selector

**URL Pattern**:
```
/grainscript/xbdghj                    (default mode)
/grainscript/xbdghj?mode=trish         (Trish B1)
/grainscript/xbdghj?mode=helen         (Helen Atthowe)
/grainscript/xbdghj?mode=davinci       (cipher)
/grainscript/xbdghj?mode=ariana        (vulnerable strength)
```

**UI Element**: Dropdown or tabs at top of grain allowing mode switching

## Why This Matters

**Accessibility**: Different people learn differently. Farmers learn through soil metaphors. Artists through emotional connection. Scholars through references. Grainmodes meet learners where they are.

**Protection**: Da Vinci cipher mode protects patent language while keeping truth visible to legitimate examiners.

**Connection**: Each mode has appropriate contact paths, turning grains into networking opportunities.

**Pedagogy**: The mode itself teaches. Seeing the same concept through multiple lenses deepens understanding.

**Beauty**: Like a musical theme with variations, grainmodes create artistic richness while maintaining structural integrity.
