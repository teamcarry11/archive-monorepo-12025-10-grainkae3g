## How to Read This Grainbook

**As Physical Book**:
- Print in monospace font (Courier New, 10pt)
- Each grain is one page (or two pages facing)
- Read sequentially 1-5
- Keep as reference for ecological-technical parallels

**As Digital Cards**:
- Read on any device with monospace display
- Terminal, e-ink reader, phone, tablet
- Works offline (markdown files)
- Share individual letters as needed

**As Gift**:
- Mail to veganic farmers learning technology
